# 📊 Superstore SQL Data Analysis Project

## 🔍 Project Overview

This project performs an end-to-end analysis of the Superstore dataset using SQL.
The objective is to extract meaningful business insights related to sales performance, customer behavior, profitability, and regional trends.

The project is structured into three levels — Basic, Intermediate, and Advanced — demonstrating progressive mastery of SQL concepts.

---

## Skills & Concepts Applied

* SQL Querying (SELECT, WHERE, GROUP BY, HAVING)
* Joins and Aggregations
* Window Functions:

  * `RANK()`
  * `LAG()`, `LEAD()`
  * `SUM() OVER()`
* Data Analysis & Business Problem Solving

---

# Project Structure

## Level 1: Basic Analysis

* Data filtering and sorting
* Basic aggregations (SUM, COUNT, AVG)
* Understanding dataset structure

## Level 2: Intermediate Analysis

* Group-based insights (region, category, customer)
* Identifying high sales but low profit scenarios
* Customer segmentation and performance analysis

## Level 3: Advanced Analysis

* Ranking top customers by profitability using window functions
* Month-over-Month sales trend analysis using `LAG()`
* Category contribution to regional sales
* Customer order frequency analysis
* Cumulative sales and profit trends
* Identifying profit margin outliers

---

## Key Business Insights

* Identified top-performing customers in each region based on profitability
* Discovered customers contributing to overall losses
* Analyzed sales growth trends across regions over time
* Evaluated product category contributions to total revenue
* Detected unusual profit margins and potential outliers
* Assessed customer purchasing behavior and frequency

---

## Tools & Technologies

* SQL (MySQL / PostgreSQL)
* Dataset: Superstore Sales Data

## How to Use

1. Download the dataset from the `/dataset` folder
2. Import into your SQL environment
3. Run queries from:

   * `level_1_basic_analysis.sql`
   * `level_2_intermediate_analysis.sql`
   * `level_3_advanced_analysis.sql`
